## ✨ Aperçu

**MarketFlow** est une application fullstack permettant :

* 📦 Gestion des produits
* 🗂️ Gestion des catégories
* 📊 Suivi d’inventaire
* 👤 Gestion de profil
* 🔐 Authentification sécurisée

---

## 🧠 Philosophie

> **Keep UI pure. Push logic to the server.**

* UI = composants simples et réutilisables
* Server = logique métier centralisée
* Types = sécurité maximale

---

## 🧰 Stack technique

| Technologie     | Rôle    |
| --------------- | ------- |
| React 18        | UI      |
| React Router v7 | Routing |
| TypeScript      | Typage  |
| Vite            | Build   |
| Supabase        | Backend |
| Tailwind CSS    | UI      |

---

## 🏗️ Architecture

* 🔒 `*.server.ts` → logique backend
* 🎯 `routes/` → organisation par feature
* 🧩 `components/` → UI découplée
* 🧠 `services/` → logique métier
* 🧾 `types/` → typage global

---
